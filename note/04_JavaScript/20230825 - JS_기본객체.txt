<JavaScript_기본객체>
*자바에서는 기본 자료형과 객체자료형이 완벽하게 분리
*자바스크립트에서의 기본데이터타입(number, string, boolean) vs 객체데이터타입
 -let priNum = 237; // 기본 데이터타입
 -let objNum = new Number(273); // 객체 데이터타입
*Object객체:자바스크립트의 최상위 객체
  -­Object 객체는 모든 자바 스크립트 객체의 최상위 객체이므로, 
   Object 객체의 프로토타입에 속성 또는 메서드를 추가하면 모든 객체에서 활용할 수 있다.

1.Number:숫자 객체
2.String:문자열을 다루기 위한 객체
3.Array:배열을 만들어 주는 객체
4.Date:날짜와 시간을 처리하기 위한 객체
5.Math:수학 계산을 위한 객체
