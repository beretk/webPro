1.이벤트란?
-이벤트란 키보드를 이용해 버튼을 입력하거나 마우스 클릭과 같이 다른 것에 영향을 미치는 것.
-사전적 정의 : 자바 스크립트에서 DOM을 통해 동적인 화면처리를 위해 처리하는 내용
:마우스 이벤트, 키보드 이벤트, HTML 프레임 이벤트, HTML 입력 양식 이벤트, 유저 인터페이스 이벤트, 구조 변화 이벤트, 터치 이벤트

*window.onload = function(){ 이벤트는 반드시 여기(function) 안에}
 1)인라인 이벤트 모델
 2)기본 이벤트 모델
 -(고전 이벤트 모델) : 이벤트 속셩에 function을 할당
 -on으로 시작하는 속성 : 이벤트 속성 ex.onclick
 -이벤트 이름 : click
 -이벤트 속성 변수 : onclick
 -이벤트 리스너 함수 : whenClick
 3)표준 이벤트 모델(*)
 4)this.html
 -클릭이벤트가 발생된 컴포넌트 객체. this 사용시 화살표 함수 사용불가
 5)이벤트 강제 발생

2.디폴트 이벤트 제거
-디폴트 이벤트
:일부 HTML 태그는 이미 이벤트 리스너가 있다. 이러한 것을 디폴트 이벤트라 한다. 
(예를들어 a태그를 클릭하면 다른 페이지로 이동하는 것.)
-이벤트 전달:어떠한 이벤트가 먼저 발생해 어떤 순서로 발생하는지를 이벤트 전달이라 한다. 
 자바스크립트는 자식 노드에서 부모 노드 순으로 이벤트를 실행한다. 
-디폴트 이벤트 제한 : 인라인 이벤트 모델
