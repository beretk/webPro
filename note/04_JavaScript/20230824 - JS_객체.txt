<JavaScript_객체>
1.속성과 메소드
-배열 내부에 있는 값:요소(element) / 객체 내부에 있는 값:속성(property)
:자바스크립트에서는 요소와 속성을 구분하지 않고 사용
*메소드 내에서 this 키워드 생략불가
2.객체와 반복문
-배열은 단순 for 반복문과 for in 반복문으로 요소에 쉽게 접근. 
-객체는 단순 for문은 불가. for in반복문을 사용한다
3.생성자 함수 개요
-생성자는 객체지향 프로그램에서 객체를 생성할 때 호출되는 단위 메서드를 말한다.
-일반함수:소문자로 시작
-생성자함수:대문자로 시작
:생성자함수는 반드시 new로 호출(new 키워드로 함수를 호출하면 객체를 위한 공간을 만들고 this 키워드가 해당 공간을 의미한다.
 그렇지 않는 경우, 일반적으로 this 키워드를 사용하면 window 객체를 나타낸다. Window 객체에 속성을 추가한 것이 된다.)
-프로토타입이란, 생성자 함수로 생성된 객체가 공통으로 가지는 공간.
