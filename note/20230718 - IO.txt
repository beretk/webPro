<I/O>자바 입/출력

1.입/출력이란?
-데이터를 읽는다(Input),데이터를 쓴다(Output)
-Input 
:파일 데이터를 읽는다.키보드의 데이터를 읽는다.네트워크상의 데이터를 읽는다(전송)
-Output
:파일에 데이터를 쓴다.모니터에 데이터를 쓴다(출력).네트워크상에 데이터를 쓴다(전송)

2.Stream이란?
-데이터를 운반(입/출력)하는데 주로 사용되는 연결 통로.
-하나의 Stream으로 입/출력을 동시에 수행 할수 없음(단방향 통신)

3.자바의 파일 입출력API
 (1)InputStream : 1byte단위 입력.이미지,동영상 등의 자료에 주로 사용
 (2)OutputStream : 1byte단위 출력.이미지,동영상 등의 자료에 주로 사용
 (3)Reader : 2byte단위 입력.문자열에 주로 사용
 (4)Writer : 2byte단위 출력.문자열에 주로 사용

4.예외 처리와 무조건 close()실행
-반드시 해야 하는 예외처리 : IOException
-작업의 마지막은 반드시 close()로 외부 연결을 끝내야 함

5.보조 Stream : 속도를 빠르게 하기 위한 보조 스트림
-BufferedReader / BufferedWriter
-PrintWriter

*스트림 순서에 주의
-스트림 생성(기본 스트림 -> 보조 스트림)
-스트림 닫을때(보조 스트림 -> 기본 스트림)
