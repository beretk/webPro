<SQL문의 종류>
1.DML : Date Manipulation Language(SELECT, INSERT, UPDATE, DELETE)
-INSERT : 데이터베이스 객체에 데이터를 입력(INSERT INTO 테이블명 VALUES (값1, 값2, ...);)
-DELETE : 데이터베이스 객체에 데이터를 삭제
-UPDATE : 기존에 존재하는 데이터베이스 객체 안의 데이터수정(UPDATE 테이블 SET 필드명1=값1, 필드명2=값2,... [WHERE 조건];)
-SELECT : 데이터베이스 객체로부터 데이터를 검색
-PRIMARY KEY : 제약조건 (UNIQUE, NOT NULL)

2.DDL : Data Definition Language(테이블 생성, 테이블 삭제, 데이블 구조 변경, 테이블의 모든 데이터제거)
-CREATE : 데이터 베이스 객체 생성(테이블생성, 테이블 구조를 정의)
-ALTER : 기존에 존재하는 테이블의 구조를 변경
	-ADD : 필드추가
	-MODIFY : 필드수정
	-DROP : 필드삭제(데이터 베이스 객체를 삭제) 
- DROP TABEL : 테이블 삭제
-TRUNCATE : 테이블의 모든행 제거(ROLLBACK 사용불가)
-RENAME : 테이블명 변경

3.DCL : Date Control LanguageDCL(사용자계정 생성, 사용자에게 권한부여, 권한박탈, 사용자계정 삭제, 트랜젝션명령어)
-GRANT : 데이터 베이스 객체에 권한 부여
-REVOKE : 이미 부여된 데이터베이스 객체의 권한을 취소
-COMMIT : 트랜잭션 확정 (TCL)
-ROLLBACK : 트랜잭션 취소 (TCL)
-SAVEPOINT : 복귀지점 설정 (TCL)
