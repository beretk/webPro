<함수 - 단일행 함수>
**단일행함수 : 숫자관련함수, 문자관련함수(1998/12/** ,홍**...), 날짜관련함수, 형변환함수, NVL(), ETC...**

1. 숫자관련함수 : 숫자를 처리하는 함수
-ABS : 절대값
-FLOOR : 소수점 아래를 버림
-ROUND : 특정 자릿수에서 반올림
-ROUND(데이터, 반올림할 소수점 자리수) : 반올림할 자리수가 음수인 경우 -1은 십단위. -2는 백단위
-TRUNC(x, n) : n으로 지정한 자리수 이하를 버림
-MOD(x, n) : x를 n으로 나눈 나머지값

2. 문자관련함수
-LOWER(str) : 소문자로
-UPPER(str) : 대문자로
-INITCAP(str) : 첫 글자만 대문자로 나머지 글자는 소문자로
-CONCAT(str1, str2) : 문자 연결 ||
-SUBSTR(str, 시작위치, 문자갯수) : STR을 시작위치부터 문자갯수만큼 추출, 시작할위치는 인덱스 아님
	시작할 위치가 음수이면 끝에서부터 자리수를 센다
-SUBSTRB(str, 시작할위치,추출할갯수) : 문자를 잘라 추출 (한글은 3byte)
-LENGTH(str) : STR의 글자수 (한글은 1byte)
-LENGTHB(str) : STR의 바이트 수(한글은 2~3byte)
-INSTR(str, 찾을글자, 시작위치) STR에서 첫번째문자부터 찾을문자가 나오는 위치(없으면 0)
-LPAD, RPAD : ( STR, 자리수, 채울문자) 입력 받은 문자열과 기호를 정렬하여 특정 길이의 문자열로 반환한다
	LPAD(왼쪽 빈자리에 채울문자) / RPAD(오른쪽 빈자리에 채울문자)
-TRIM : 여백제거//TRIM(STR),LTRIM(STR),RTRIM(STR)
-REPLACE : STR열에서 바꿔야할 문자를 바뀔문자로 교체

3. 날짜관련함수
-SYSDATE : 지금
-MONTHS_BETWEEN(시점1,시점2) : 두 시점간의 개월수 (시점1이 나중시점)
-ADD_MONTHS(특정시점,개월수) : 특정시점부터 몇 개월후
-NEXT_DAY(특정시점,'토') : 특정시점부터 처음 도래하는 토요일
-LAST_DAY(특정시점) : 특정시점의 말일(28,29,30,31)
-ROUND(날짜, XX) : 날짜를 반올림(XX:YEAR,MONTH,DAY)
	TRUNCS(날짜,XX) : 날짜 내림

4. 형변환함수
-TO_CHAR(날짜형, '출력형식') / TO_CHAR(숫자형, '출력형식')
:출력형식 0 : 자리수.출력형식의 자릿수가 숫자형데이터 보다 많으면 0으로 채움
         9 : 자리수.출력형식의 자릿수가 숫자형데이터 보다 많아도 숫자형데이터만 출력
         , :
         L : 지역통화단위를 추가
         $ : 통화단위 $추가
-TO_DATE(문자, '패턴')
-TO_NUMBER(문자, '패턴')
